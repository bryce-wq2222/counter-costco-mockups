Notes for review & build.

TYPEFACES ARE STAND-INS

Jost stands in for Geograph Bold, Space Mono for GT Flexa Mono, and Kumbh Sans for Supreme. The production build swaps in the licensed brand fonts one-for-one.

MAP IS A PLACEHOLDER

The Southeast map and pins mock the intended behavior. In production this area embeds the live club locator, wired to real club data and distances.

FORMS ARE FRONT-END ONLY

Zip, email, and phone fields validate inline and demo the full signup flow, but nothing is submitted anywhere. Production connects to the email/SMS platform and the $5-back rebate flow.

SELF-CONTAINED SPEC

Each mockup is a single HTML file with all styles, interactions, and imagery inside — it doubles as the implementation reference for the Shopify build.
